# 🧩 Daily Card Template (for building new days)

> **For the AI agent / Kiro:** copy this exact shape for every new day card. Keep it
> short, concrete, and linked. One card = one day = ~4 hours (2h learn + 2h practice).
> See [`../AGENT-GUIDE.md`](../AGENT-GUIDE.md) for the rules.

---

## Week _N_ · Day _N_ — _Short Title_

🎯 **Today:** _One sentence — what this day is about._

⏱️ **Morning (~2h) — Learn**
- ▶️ _Resource name_ — [link](https://example.com) — _what to watch/read (e.g., "Chapter 1, ~40 min")_
- 📝 As you go, jot 3–5 notes in your journal repo, in your own words.

🛠️ **Night (~2h) — Practice (do ONE thing)**
- _A single, concrete, doable task that produces something you can commit._
- _Optional stretch task if energetic._

✅ **Checklist**
- [ ] Watched/read today's resource
- [ ] Wrote notes in my own words
- [ ] Did the practice task
- [ ] Committed today's work to GitHub
- [ ] Can explain today's idea to a friend in 2 minutes

📝 **Log:** add one line to [`../PROGRESS.md`](../PROGRESS.md).

➡️ **Next:** _one line teaser of tomorrow._

---

### Notes for the builder
- Use resources from [`best-resources.md`](./best-resources.md) first.
- ~1 tool day per week (from [`tools.md`](./tools.md)).
- If a topic is big, split across days — never cram.
- Always use real links; if unsure, tell the learner to search the exact title.
