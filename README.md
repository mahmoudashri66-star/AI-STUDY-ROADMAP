# 🧠 My AI Learning Plan

A simple, day-by-day plan to learn AI **deeply** — from zero to building real AI apps
and agents — while finally understanding the tools I already use (Kiro, GitHub, z.ai,
Supabase, Cloudflare, and more).

No overwhelm. Every day I just do this:

> ### ☀️ Open → ▶️ Learn → 🛠️ Practice → ✅ Tick the boxes → 😴 Done → Tomorrow

---

## 👉 How to use this every day

1. Open **[`START-HERE.md`](./START-HERE.md)**.
2. It tells me which **Day** I'm on.
3. I open that day's card, click the links, watch/read, do the one practice task.
4. I tick the checkboxes and write one line in **[`PROGRESS.md`](./PROGRESS.md)**.
5. Close the laptop. Come back tomorrow. That's it.

**One rule:** if I can't explain today's idea to a friend in 2 minutes, I redo the
practice tomorrow before moving on. Understanding > speed.
