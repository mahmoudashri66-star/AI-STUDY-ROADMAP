🎯 **Today:** set up access to call an LLM from your own code — and understand exactly what an API key is.

💡 **Cost note:** API usage costs a little (often cents). Set a spending cap in the provider dashboard. $0 option: use **Ollama** (free local models) — noted on Day 168.

☀️ **Morning (~2h) — Learn**
- ▶️ Read the "z.ai" and "API key" sections of your [tools guide](https://github.com/mahmoudashri66-star/AI-STUDY-ROADMAP/blob/main/plan/tools.md).
- ▶️ Skim provider docs: [z.ai](https://docs.z.ai/guides/overview/quick-start) or [OpenAI](https://platform.openai.com/docs) / [Anthropic](https://docs.anthropic.com).
- 📝 Note: an API key = a secret password that lets your code use the model (and bills you). Never commit it!

🌙 **Night (~2h) — Practice**
- Create an API key with one provider. Set a low spend limit. Store the key safely (like your bot secrets — NOT in code).
- Commit a note (not the key!): `Day 163: API access ready`.

✅ **Checklist**
- Understood API keys
- Created a key + set a spend cap
- Stored it safely (never in the repo)
- I'm ready to call an LLM from code
