🎯 **Today:** learn what an LLM *actually* reads — not words, but **tokens** (chunks of text, often subwords).

☀️ **Morning (~2h) — Learn**
- ▶️ Hugging Face LLM course — tokenizers intro, or search "what is a token LLM".
- 📝 Note: a token might be a word, part of a word, or punctuation. Models see token IDs, not letters.

🌙 **Night (~2h) — Practice**
- Try OpenAI's Tokenizer page (or `tiktoken` in Colab): paste a sentence and see how it splits into tokens. Count them.
