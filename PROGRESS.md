## Phase Checkpoints

### Phase 0 — Orientation _(target ~2 weeks)_
- [ ] Can explain how AI/ML/DL/LLM/agent nest together
- [ ] Can define token, prompt, context window, training, inference, parameter, model
- [ ] Can run Python in Colab and commit to GitHub
- [ ] `glossary.md` + `mental-model.md` committed

### Phase 1 — Python Foundations _(target ~8 weeks)_
- [ ] Can read a traceback from my own code and fix the bug unaided
- [ ] Can write a program with functions + loop + file I/O from a blank file
- [ ] Can explain list vs. dictionary and when to use each
- [ ] Capstone CLI tool runs, in its own repo with a README I wrote
- [ ] Can read an unfamiliar snippet and describe what it does

### Phase 2 — Machine Learning _(target ~8 weeks)_
- [ ] Can explain how a model learns (predict → loss → adjust → repeat)
- [ ] Can explain and demonstrate overfitting in code
- [ ] Can train a scikit-learn model with train/test split and proper metrics
- [ ] Capstone repo with honest "what it learned / limitations" writeup
- [ ] Understand why high accuracy can still be a bad model

### Phase 3 — Deep Learning & LLMs _(target ~10 weeks)_
- [ ] Can explain forward pass and backpropagation plainly
- [ ] Can explain token, embedding, attention, next-token prediction
- [ ] Built a tiny neural net by hand + re-implemented in PyTorch
- [ ] Completed Karpathy tiny-GPT build (runs)
- [ ] `how-llms-work.md` clear enough to teach a beginner

### Phase 4 — LLM Apps, APIs, RAG _(target ~8 weeks)_
- [ ] Can make a raw LLM API call and explain the full request/response
- [ ] Can explain + demonstrate the full RAG pipeline
- [ ] RAG app answers over my own documents behind a simple UI
- [ ] Can describe 3 prompt techniques and why they work
- [ ] Capstone public on GitHub with limitations writeup

### Phase 5 — AI Agents _(target ~8 weeks)_
- [ ] Built an agent loop from scratch (no framework) with tools
- [ ] Can explain reason–act loop, tool selection, memory, planning
- [ ] Rebuilt an agent in a framework + can explain what it abstracts
- [ ] Can explain why a coding agent like Kiro decides what it decides
- [ ] Capstone agent completes a real multi-step task with guardrails (public)

### Phase 6 — Shipping & Architecture _(target ~8 weeks+)_
- [ ] Have a live, public, deployed AI product
- [ ] Can manage secrets, track cost, handle latency/errors
- [ ] Can measure app quality with an evaluation set
- [ ] Can write an architecture doc reasoning about tradeoffs
- [ ] Can decide what AI a new problem needs + evaluate a new tool/model
